# Seattle Dinner Finder

Daily curated restaurant recommendations for north Seattle neighborhoods: Ballard, Fremont, Green Lake, and Queen Anne.

## How it works

Every morning at 9 AM PT, an AI agent researches fun restaurants with available reservations for that evening. Results are published to this GitHub Pages site.
